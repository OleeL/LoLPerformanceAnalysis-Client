# Requirements

* [Node JS + NPM](https://nodejs.org/en/)
* [LolPerformanceAnalysisAPI](https://github.com/OleeL/LoLPerformanceAnalysis-API)

## Download packages
```
npm install
```

## Run in release mode
```
npm run build
npm run start
```

## Run in development mode
```
npm run dev
```
